ARMANDA - 3D Talking Agent v0.5
by Patrizio Migliarini

When the Talk button is pressed, the text in the texbox below (or, if no input given, a server-side file) is parsed and the model is animated, the text is also passed to the Text-to-Speech API for the audio to be rendered. The expression of this model can also be changed utilizing the sliders in the control panel. When loading, a test is done to verify if the browser is Text-to-Speech capable and the button label is changed accordingly.

Rendered with WebGL using three.js
Text-to-Speech through Web Speech API
Speech-to-Text through Google API
Model by David Steele

To move the camera use the arrow keys.
